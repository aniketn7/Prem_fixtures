Author: Aniket Narkhede

Description: This app allows the user to query upcoming soccer matches in the English Premier League for a specific team that he/she selects. 
             The user can save/star matches that he/she is interested in and later delete them if needed.

API: https://rapidapi.com/api-sports/api/api-football/
(GET requests made to multiple endpoints of this API)

YouTube demo: https://youtu.be/30b4rTzE2gA
